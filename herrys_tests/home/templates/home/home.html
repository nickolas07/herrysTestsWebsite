Wie erstellt man sich einen Test?

Als Lehrer

  1. Sie gehen oben auf "Erstellen" und dann auf "Lehrer"
  2. Jetzt können Sie unter "Filter" nach Jahrgängen sortieren
  3. Wenn Sie sich jetzt für einen Test entschieden hat, müssen Sie die Klasse, den Kurs und den Namen der in der Überschrift stehen soll angeben
  4. Jetzt können Sie auf "Erstellen" drücken und es wird sich ein neuer Tab öffnen, es wird eine kurze Zeit laden, da der Test erstellt wird
  5. Fertig geladen, sehen Sie eine Vorschau von dem Test, jetzt können Sie sich diesen anschauen
  6. Eine Möglichkeit einen Test herunterzuladen ist, indem Sie:
    1. Wieder auf den vorherigen Tab wechseln
    2. Auf den kleinen Pfeil nach unten neben "Erstellen" drücken
    3. Jetzt können Sie sich den Test nochmal anschauen, ihn herunterladen, sowie die Lösung anschauen und herunterladen

Als Schüler

  1. Du gehst oben auf "Erstellen" und dann auf "Schüler"
  2. Jetzt kannst du unter "Filter" nach Jahrgängen sortieren
  3. Wenn du dich dann für einen Test entschieden hast, kannst du auf "Erstellen" drücken und es wird sich ein neuer Tab öffnen, es wird eine kurze Zeit laden, da der Test erstellt wird
  4. Fertig geladen, siehst du eine Vorschau von dem Test, jetzt kannst du dir diesen anschauen
  5. Eine Möglichkeit einen Test herunterzuladen ist, indem du:
    1. Wieder auf den vorherigen Tab wechselst
    2. Auf den kleinen Pfeil nach unten neben "Erstellen" drückst
    3. Jetzt kannst du dir nochmal den Test anschauen, ihn herunterladen, sowie die Lösung anschauen und herunterladen
